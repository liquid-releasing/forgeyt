# Archive — early design specs (2026-03-25)

These markdown files are pre-implementation design thinking written
before ForgeYT was actually built. In several places they describe
architectures and scope that differ from what shipped in `v0.0.1-alpha`.

They are preserved here for historical context only. **The actual
implementation is in the top-level source tree** (`forgeyt.py`, `ui.py`,
`forgeyt_core/`, `ForgeYT.spec`), and the user-facing truth is in the
top-level `README.md`.

Do not use these documents as reference for current behavior.
